Installation Guide

Follow the following steps to install the Track Machine app. This has been tested for Ubuntu only.

  • 1. Download the zip file
  • 2. Unzip using tar/unzip command
  • 3. cd machine_track
  • 4. Run install.sh (./install.sh)
  • 5. Create / Edit crontab file.

If you have an issues on this, please raise a ticket in our help page

Create Crontab

Follow the following steps to install create the Crontab File.

  • 1. When the text editor gets opened copy the following line
  • 2. 22 18 * * * /bin/bash /home/(username)/run_python.sh
  • 3. You can get the username by running whoami.
  • 4. Save and Close.

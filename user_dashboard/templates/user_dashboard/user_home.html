Installation Guide

Follow the following steps to install the Track Machine app. This has been tested for Ubuntu only.

  • 1. Download the zip file
  • 2. Unzip using tar/unzip command
  • 3. cd machine_track
  • 4. Run install.sh (./install.sh)
  • 5. Create crontab file.For more information please see Create Crontab in Linux

If you have an issues on this, please raise a ticket in our help page

Features
  • 1. Can set passwords for your frequent login pages.
  • 2. Can get your browser history.
  • 3. Can get your top sites.
  • 4. Can get your bookmarks.
  • 5. Can get your machine details.
